# Listing Assets

This folder contains repo-side Shopify listing assets for VedaSuite AI.

## Files

- `vedasuite-app-icon.svg`
  - premium app icon artwork
  - suitable as a master vector source for exported PNG sizes
- `vedasuite-app-screenshot-dashboard.svg`
  - polished dashboard screenshot-style marketing asset
  - suitable as a source for exported App Store screenshots
- `vedasuite-app-screenshot-fraud.svg`
  - fraud intelligence listing screenshot
- `vedasuite-app-screenshot-pricing.svg`
  - pricing strategy listing screenshot
- `vedasuite-app-screenshot-subscription.svg`
  - subscription plans listing screenshot
- `vedasuite-app-screenshot-competitor.svg`
  - competitor intelligence listing screenshot
- `vedasuite-app-screenshot-reports.svg`
  - reports and weekly intelligence listing screenshot
- `vedasuite-app-screenshot-settings.svg`
  - merchant controls and settings listing screenshot

## Recommended exports

- App icon: `1200x1200` PNG
- Screenshots:
  - `1600x1200` PNG for Shopify listing
  - optional additional crops for press kit and marketing pages

## Notes

These assets are original repo-side source artwork. Export PNG versions before uploading to Shopify Partner Dashboard.
