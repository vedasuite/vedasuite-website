# VedaSuite Website Wireframe Copy

Use this document as a direct prompt or source brief for an AI website builder.

The site should feel premium, intelligent, calm, and high-trust. Avoid generic SaaS design. The visual language should feel like a luxury commerce operating system built for serious Shopify merchants.

## Brand Direction

- Brand name: `VedaSuite AI`
- Tagline: `AI commerce intelligence suite for Shopify merchants`
- Tone: premium, confident, operational, sharp, trustworthy
- Audience:
  - Shopify founders
  - ecommerce operators
  - growth and revenue teams
  - fraud, pricing, and retention decision-makers
- Design references:
  - strong editorial hierarchy
  - luxury software feel
  - warm neutrals with dark high-contrast hero sections
  - subtle gold, olive, plum, and midnight accents
- Avoid:
  - startup cliches
  - playful illustrations
  - generic blue SaaS look
  - crowded dashboards everywhere

## Global Design Instructions

- Use a bold premium hero with strong typography and a spacious layout.
- Create a differentiated left-to-right narrative flow instead of stacking generic SaaS blocks.
- Keep copy crisp, but not overly minimal.
- Use rich cards, visual panels, data ribbons, product previews, and section separators.
- The site should feel handcrafted and deliberate.
- Primary CTA: `Book a demo`
- Secondary CTA: `See the platform`

## Site Map

- Home
- Features
- How It Works
- FAQ
- Early Access
- Optional footer links:
  - Privacy Policy
  - Terms of Service
  - Support
