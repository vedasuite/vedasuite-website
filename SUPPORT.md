## Support Scope

VedaSuite AI support covers:

- installation and embedded app access
- billing and subscription issues
- webhook and sync troubleshooting
- module availability and plan gating
- bug reports and unexpected product behavior

## Recommended Support Workflow

When reporting an issue, include:

- store domain
- timestamp of the issue
- request ID if shown in an error response
- screenshot or screen recording
- the module and action being attempted

## Escalation

Critical issues should be escalated when they affect:

- installation
- billing
- uninstall/compliance handling
- data sync
- data loss or privacy handling
